# zitadel_action_execution (Resource)

Resource representing an execution, which defines when and which targets are executed based on triggers.

## Example Usage

```terraform
resource "zitadel_action_execution" "default" {
  execution_type = "events"

  event = "user.human.added"
  group = "user"

  targets = [
    data.zitadel_action_target.default.id
  ]
}
```

## Schema

### Required

- `execution_type` (String) The type of execution trigger. One of: events, request, response, function.
- `targets` (List of String) The ordered list of target IDs to execute.

### Optional

- `all` (Boolean) Whether to trigger on all events in the group (for events, request and response types).
- `event` (String) The event name to trigger on (for events type).
- `function` (String) The function name to trigger on (for function type).
- `group` (String) The event group (for events type).
- `method` (String) The gRPC method name (for request/response type).
- `service` (String) The gRPC service name (for request/response type).
