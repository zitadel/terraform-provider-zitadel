# zitadel_org_idp_apple (Resource)

Resource representing an Apple IDP on the organization.

## Example Usage

```terraform
resource "zitadel_org_idp_apple" "default" {
  org_id               = data.zitadel_org.default.id
  name                 = "Apple"
  client_id            = "com.example.app"
  team_id              = "ABCDE12345"
  key_id               = "FGHIJ67890"
  private_key          = "*****abc123"
  scopes               = ["name", "email"]
  is_linking_allowed   = false
  is_creation_allowed  = true
  is_auto_creation     = false
  is_auto_update       = true
  auto_linking         = "AUTO_LINKING_OPTION_USERNAME"
}
```

### Required

- `is_creation_allowed` (Boolean) enable if users should be able to create a new account in ZITADEL when using an external account

- `org_id` (String) ID of the organization. If not provided, the organization of the authenticated user/service account is used.
