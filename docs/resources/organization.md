# zitadel_organization (Resource)

Resource representing an organization in ZITADEL, which is the highest level after the instance and contains several other resource including policies if the configuration differs to the default policies on the instance.

## Example Usage

```terraform
resource "zitadel_organization" "default" {
  name = "terraform-test"
}

resource "zitadel_organization" "with_admins" {
  name = "terraform-test-with-admins"
  admins = [
    {
      user_id = "123456789012345678"
      roles   = ["ORG_OWNER"]
    }
  ]
}

resource "zitadel_organization" "with_custom_id" {
  name   = "terraform-test-custom"
  org_id = "custom-org-id-123"
}
```
