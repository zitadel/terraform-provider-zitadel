# zitadel_organization_metadatas (Data Source)

Datasource representing metadata entries of an organization in ZITADEL.

## Example Usage

```terraform
  organization_id = "123456789012345678"
}

  organization_id = "123456789012345678"
  key             = "example"
}

output "all_metadata" {
  value = data.zitadel_organization_metadatas.default.metadata
}
```
